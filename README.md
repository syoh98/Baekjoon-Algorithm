### 📋 2. 시간복잡도
- 상수형: O(1)
- 로그형: O(logN)
- 선형: O(N)
- 선형로그형: O(NlogN)
- 2차형: O(N²)
- 3차형: O(N³)
- 지수형: O(2^N)
- 팩토리얼형: O(N!)
